# CortexFlow AI

An adaptive Data Science Language Model (DSLM) designed for industrial AI workflows.

## Features

- Dataset Profiling
- Automated Task Detection
- Dynamic Workflow Generation
- Automated Model Selection
- Explainable AI using SHAP
- Workflow Memory Repository

## Tech Stack

- Python
- Scikit-Learn
- XGBoost
- SHAP
- Streamlit
- Pandas
- NumPy
